# Project Spec: AI Documentation Assistant

- **Project:** An AI-powered Q&A assistant for a documentation website.
- **Purpose:** To provide a beginner-friendly, end-to-end example of a RAG (Retrieval-Augmented Generation) application.
- **Core Components:**
    1.  **Docusaurus Frontend:** A static site for hosting documentation content.
    2.  **FastAPI Backend:** A Python server to handle AI logic and data processing.
    3.  **RAG Pipeline:** Uses OpenAI models to understand questions and generate answers based on document content.
    4.  **Qdrant Vector DB:** Stores document embeddings for efficient similarity search.
    5.  **Chat Widget:** A simple HTML/CSS/JS interface for users to interact with the assistant.
